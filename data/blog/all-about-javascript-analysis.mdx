### JavaScript Analysis

#### Blogs & Writeups

- [Static Analysis of Client-side JavaScript for Pen Testers and Bug Bounty Hunters](https://blog.appsecco.com/static-analysis-of-client-side-javascript-for-pen-testers-and-bug-bounty-hunters-f1cb1a5d5288)
  - A comprehensive guide on performing static analysis of JavaScript code, beneficial for penetration testers and bug bounty hunters.
- [JavaScript Analysis for Pentesters](https://kpwn.de/2023/05/javascript-analysis-for-pentesters/)
  - Detailed insights into JavaScript analysis techniques tailored for penetration testers.
- [jsluice JavaScript Gold Mining - Part 1](https://bishopfox.com/blog/jsluice-javascript-gold-mining)
  - An introduction to using jsluice for mining valuable information from JavaScript files.
- [jsluice JavaScript Technical Deep Dive - Part 2](https://bishopfox.com/blog/jsluice-javascript-technical-deep-dive)
  - A deeper technical exploration of jsluice and its capabilities.
- [Hacking JavaScript with JavaScript](https://jsoverson.medium.com/hacking-javascript-with-javascript-6adbeaba22e9)
  - Techniques and strategies for hacking JavaScript using JavaScript itself.
- [JavaScript language made for bugs workshop by YesWeHack](https://www.yeswehack.com/learn-bug-bounty/javascript-language-made-for-bugs)
  - A workshop by YesWeHack focusing on the inherent vulnerabilities in JavaScript.
- [JavaScript from Sourcemaps](https://pulsesecurity.co.nz/articles/javascript-from-sourcemaps)
  - How to analyze and extract useful information from JavaScript sourcemaps.
- [Bug Bounty Hunter Guides: JavaScript Files by Zseano](https://www.bugbountyhunter.com/guides/?type=javascript_files)
  - Guides and tutorials on analyzing JavaScript files for bug bounty hunting.
- [Scanning JS files for endpoint and secrets](https://securityjunky.com/scanning-js-files-for-endpoint-and-secrets/)
  - Methods for scanning JavaScript files to uncover endpoints and hidden secrets.
- [JavaScript Enumeration for Bug Bounty Hunters by thehackerish](https://thehackerish.com/javascript-enumeration-for-bug-bounty-hunters/)
  - Techniques for enumerating JavaScript files to find vulnerabilities, specifically for bug bounty hunters.
- [Static JavaScript Analysis with Burp Suite](https://statuscode.ch/2015/05/11/static-javascript-analysis-with-burp/)
  - Utilizing Burp Suite for static analysis of JavaScript code.
- [JavaScript Analysis Conference Talks by Lewis Ardern](https://github.com/LewisArdern/ConferenceTalks/)
  - A collection of conference talks focused on JavaScript analysis by Lewis Ardern.
- [CodeQL workshop for JavaScript](https://github.com/githubsatelliteworkshops/codeql/blob/master/javascript.md)
  - A workshop on using CodeQL for JavaScript analysis.
- [Deobfuscating / Unminifying Obfuscated Web App / JavaScript Code](https://gist.github.com/0xdevalias/d8b743efb82c0e9406fc69da0d6c6581)
  -  Some notes and tools for reverse engineering / deobfuscating / unminifying obfuscated web app code

#### Github Tools

- [Mapperplus](https://github.com/midoxnet/mapperplus)
  - A tool for mapping out JavaScript files and their dependencies.
- [Sourcemapper](https://github.com/denandz/sourcemapper)
  - A utility for mapping minified JavaScript code back to its original source code using source maps.
- [Metasecjs](https://github.com/LewisArdern/metasecjs)
  - A JavaScript analysis tool focused on security research.
- [Retire.js](https://github.com/RetireJS/retire.js)
  - A tool for scanning JavaScript code and dependencies for known vulnerabilities.
- [Jsluice](https://github.com/BishopFox/jsluice)
  - A tool for extracting valuable information from JavaScript files.
- [JSA](https://github.com/w9w/JSA)
  - A JavaScript analyzer designed for security assessments.
